Introduction to CSS

Cascading Style Sheets (CSS) allows us to style HTML elements and control how they look on the page. From colors and fonts to layout and spacing, CSS brings structure and visual appeal to a website.

Understanding the Box Model

Every element in CSS is a rectangular box made up of content, padding, border, and margin. These properties determine how much space an element takes up and how it interacts with others.

  • Content: The actual text or image inside the box.
  • Padding: Space between content and the border.
  • Border: The edge surrounding the padding and content.
  • Margin: Space outside the border, separating elements.
